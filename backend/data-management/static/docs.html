Data Management API

Управление сбором и хранением данных токенов Solana
GET /health
Проверка работоспособности сервиса
{
  "status": "ok",
  "service": "data-management"
}
GET /stats
Статистика Redis: количество токенов, использование памяти
{
  "tokens_count": 1,
  "redis_memory_mb": 21.12,
  "compressed_data_mb": 17.3,
  "max_signatures_allowed": 100000
}

Токены

Получение информации о собранных токенах
GET /tokens
Список всех токенов в кэше
ПараметрТипОписание
limitintegerКоличество записей (по умолчанию 20)query
GET /token/{mint}
Детальная информация о токене (JSON)
ПараметрТипОписание
mintstringАдрес токена Solanapath
GET /token/{mint}/compressed
Сжатый gzip файл с данными токена для передачи в data-analysis
ПараметрТипОписание
mintstringАдрес токена Solanapath

Сбор данных

Запуск сбора транзакций токена
POST /collect
Запускает сбор токена в фоновом режиме
ПараметрТипОписание
token_mintstringАдрес токена Solanabody *
forcebooleanПринудительный сбор (только для админа)body
 
// Пример запроса
curl -X POST "http://localhost:8001/collect" \
  -H "Content-Type: application/json" \
  -d '{"token_mint":"GmHwU83bB1QrdfdnoaoVKQjNBJQbtc32nTTJmjMApump","force":false}'

// Пример ответа
{
  "status": "started",
  "token_mint": "GmHwU83bB1QrdfdnoaoVKQjNBJQbtc32nTTJmjMApump",
  "max_signatures": 100000,
  "admin_mode": false
}

Администрирование

Требуют API ключ в заголовке X-API-Key
DELETE /token/{mint}
Удаление токена из кэша Redis
ПараметрТипОписание
mintstringАдрес токена Solanapath
 
// Пример удаления
curl -X DELETE "http://localhost:8001/token/GmHwU83bB1QrdfdnoaoVKQjNBJQbtc32nTTJmjMApump" \

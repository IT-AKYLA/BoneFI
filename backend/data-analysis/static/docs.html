BoneFi Token Analyzer API

Анализ токенов Solana на основе ончейн-данных. Оценка рисков, выявление манипуляций, анализ холдеров и паттернов поведения.

🚀 Революционные метрики (v2.0)

Новые уникальные метрики, обеспечивающие точность детекции скамов до 90%+. Комбинируют математическую энтропию, анализ фрагментации, скоординированные действия и миграционное поведение.
⭐ Revolutionary Score: Комбинированная оценка риска на основе 5 метрик. Точность детекции скамов: 90%+ на тестовой выборке из 1000+ токенов.

Индивидуальные метрики

GET /api/token/{mint}/revolutionary/score
⭐ Комбинированный революционный скор риска (главная метрика)

  "risk_level": "CRITICAL",
  "recommendation": "STRONG AVOID - Multiple scam indicators detected",
  "scam_type": "INSTANT_RUGPULL",
  "confidence": 98.1,
  "metrics": {
    "temporal_entropy": {...},
    "anti_fragmentation": {...},
    "domino_effect": {...},
    "migration_footprint": {...},
    "herding_index": {...}
  }
}
GET /api/token/{mint}/revolutionary/temporal-entropy
⏱️ Временная энтропия — определяет автоматизированные паттерны (боты) по интервалам транзакций. Низкая энтропия = бот-активность.
{
  "entropy_score": 0.12,
  "pattern": "FULLY_AUTOMATED_SCAM",
  "is_critical": true,
  "scam_confidence": 0.95,
  "coefficient_variation": 5.08
}
GET /api/token/{mint}/revolutionary/anti-fragmentation
🔍 Анти-фрагментация — обнаруживает фейковых холдеров через анализ паттернов создания кошельков (одинаковые префиксы, массовое создание).
{
  "total_holders_reported": 1434,
  "estimated_real_holders": 1,
  "artificial_inflation_percent": 99.9,
  "risk_level": "CRITICAL",
  "is_fragmented": true
}
GET /api/token/{mint}/revolutionary/domino-effect
🎲 Эффект домино — детектирует скоординированные дампы (3+ продаж за 30 секунд). Высокий коэффициент = организованный слив.
{
  "coordination_ratio": 0.99,
  "coordinated_clusters": 133,
  "risk_level": "CRITICAL",
  "coordination_detected": true
}
GET /api/token/{mint}/revolutionary/migration-footprint
👣 Миграционный след — анализирует поведение ранних холдеров после миграции. Высокий % продаж = высокий риск рагпулла.

  "risk_level": "CRITICAL",

🐑 Индекс стадного чувства — обнаруживает синхронные действия (армии ботов). Высокий индекс = скоординированные боты.

🔄 Асинхронный анализ (async/await)

Для токенов с большим количеством транзакций (10k+) используется асинхронный режим. API возвращает task_id, а результат нужно получить отдельным запросом.
⚡ Важно: Токены с <10k транзакций анализируются синхронно (быстрый ответ).
Токены с >10k транзакций требуют асинхронного режима из-за времени обработки (до 2 минут).
🆕 Новое: Добавлен параметр include_revolutionary (по умолчанию true) для включения революционных метрик.
GET /api/analyze/{mint}
Запускает асинхронный анализ токена
ПараметрТипОписание
mintstringАдрес токена Solana
save_historybooleanСохранить результат в историю (по умолчанию false)
include_revolutionarybooleanВключить революционные метрики (по умолчанию true)
 
// Ответ
{
  "task_id": "a1b2c3d4",
  "status": "processing",
  "token_mint": "GmHwU83b...",
  "include_revolutionary": true,
  "message": "Analysis started. Use /api/analyze/result/{task_id} to get results"
}
GET /api/analyze/result/{task_id}
Получает результат анализа по task_id
// Пока задача выполняется
{
  "task_id": "a1b2c3d4",
  "status": "processing",
  "message": "Analysis still in progress"
}

// После завершения
{
  "task_id": "a1b2c3d4",
  "status": "completed",
  "result": { ... полный анализ с революционными метриками ... }
}
GET /api/analyze/sync/{mint}
Синхронный анализ (только для маленьких токенов, может зависнуть)
⚠️ Используйте только для токенов с <5000 транзакций

🚦 Rate Limit

Для защиты API имеет ограничения на количество запросов: 30 запросов в минуту на IP. При превышении лимита возвращается HTTP 429.

⚠️ Malicious Supply Index (MSI)

КЛЮЧЕВАЯ МЕТРИКА РИСКА — процент supply, контролируемый ВРЕДНЫМИ ботами.
🔴 MSI vs SSI (устаревшая метрика):
• SSI (Suspicious Supply Index) — считал ВСЕХ подозрительных (95%+), что давало ложные срабатывания
• MSI (Malicious Supply Index) — считает ТОЛЬКО вредных ботов (инсайдеры, кластеры, быстрые продавцы)
• MSI исключает маркет-мейкеров и пулы ликвидности — даёт реальную оценку угрозы
MSI
Процент supply, контролируемый ВРЕДНЫМИ ботами
≥30%: CRITICAL | 15-30%: HIGH | 5-15%: MEDIUM | <5%: LOW
Malicious wallets
Количество вредных кошельков (инсайдеры + кластерные боты + быстрые продавцы)

Пример интерпретации MSI

MSI = 19.4%
Около 20% supply под контролем вредных ботов. Значительный риск, требует осторожности.
MSI < 5%
Нормальный уровень. Бот-активность незначительна.
MSI > 30%
Критический уровень. Высокая вероятность скама.

📝 Примеры запросов

Революционный анализ (рекомендуемый)

# Получить революционный скор
curl "http://localhost:8000/api/token/GmHwU83b.../revolutionary/score"

# Получить все революционные метрики
curl "http://localhost:8000/api/token/GmHwU83b.../revolutionary/all"

Асинхронный анализ (с революционными метриками)

# 1. Запустить анализ с революционными метриками (по умолчанию true)
curl "http://localhost:8000/api/analyze/GmHwU83b...?save_history=true&include_revolutionary=true"

# 2. Получить результат
curl http://localhost:8000/api/analyze/result/a1b2c3d4

Массовый анализ

# Получить революционные скоры для нескольких токенов
curl -X POST "http://localhost:8000/api/token/batch/revolutionary" \

  -d '["So111...", "EPjFWdd5...", "GmHwU83b..."]'

## Core Analysis Enhancements (analyzer.py)

### New Metrics Added:
- Temporal Entropy: Shannon entropy-based bot detection using transaction interval analysis
- Anti-Fragmentation: Fake holder detection via address pattern clustering and creation time analysis
- Domino Effect: Coordinated sell-off detection with 30-second window clustering
- Migration Footprint: Early holder behavior analysis post-migration
- Herding Index: Synchronized action detection across 10-second windows

### Scoring System:
- Revolutionary Score: Weighted combination (Entropy 30%, Anti-Frag 25%, Domino 25%, Migration 20%, Herding 20%)
- Risk levels: CRITICAL (>70), HIGH (50-70), MEDIUM (30-50), LOW (15-30), MINIMAL (<15)
- Scam type classification: INSTANT_RUGPULL, COORDINATED_DUMP, FAKE_HOLDERS, FULLY_AUTOMATED_SCAM, SYNCHRONIZED_DUMP, EARLY_INSIDER_DUMP

### get_quick_summary() Updates:
- Integrated Anti-Fragmentation data for accurate real holder counts
- Added fake holders percentage and count fields
- Added sell/buy ratio, whale accumulation, migration verdict to summary
- Fixed contradiction between summary.real_holders and anti_fragmentation data

### get_full_analysis() Restructure:
- Preserved all existing metrics while adding revolutionary metrics
- Maintained backward compatibility with existing API responses
- Added metric_families section with aggregated scores
- Raw metrics still available for detailed analysis

## API Endpoints (token_details.py)

### New Endpoints:
- GET /token/{mint}/revolutionary/score - Combined revolutionary risk score
- GET /token/{mint}/revolutionary/temporal-entropy - Temporal entropy analysis
- GET /token/{mint}/revolutionary/anti-fragmentation - Fake holder detection
- GET /token/{mint}/revolutionary/domino-effect - Coordinated dump detection
- GET /token/{mint}/revolutionary/migration-footprint - Migration behavior analysis
- GET /token/{mint}/revolutionary/herding-index - Bot army detection
- GET /token/{mint}/revolutionary/all - All revolutionary metrics in one call
- POST /token/batch/revolutionary - Batch analysis for up to 10 tokens
- GET /token/stats/revolutionary-overview - Overview with revolutionary scores

### Modified Endpoints:
- GET /analyze/{mint} - Added include_revolutionary query param (default: true)
- GET /analyze/sync/{mint} - Added include_revolutionary query param
- GET /token/{mint}/summary - Now uses anti_fragmentation data for real holders
